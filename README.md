# 🎓 UniService Marketplace

**UniService** is a premium, peer-to-peer marketplace built specifically for university students. It empowers campus life by connecting students who need help with those who have skills to share—from tutoring and moving to cleaning and creative services.

![UniService Hero Mockup](./hero-mockup.png)

## 📁 Repository Structure

The main application code is located in the [uni-service-main](./uni-service-main) directory.

- 📂 [uni-service-main](./uni-service-main): The complete Vite, React, and TypeScript codebase.
- 📂 [uni-service-main/src](./uni-service-main/src): All frontend components, hooks, and pages.

## 🚀 Quick Start

To get the project running locally:

```sh
cd uni-service-main
npm install
npm run dev
```

For more detailed information, please see the [detailed README](./uni-service-main/README.md) in the `uni-service-main` directory.
